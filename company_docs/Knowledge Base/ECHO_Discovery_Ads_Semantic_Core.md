# ECHO Discovery Ads Semantic Core - Updated, Evidence-Based
Working manual for ad managers and designers

Updated working manual for Discovery Ads. This version keeps the original emotional-first structure but re-ranks message priorities using the evidence-based TA persona and the new cross-survey KB.

## 1. North star
- Lead with nostalgia first, togetherness second, ease third, and proof-backed adult-home fit after interest exists.
- Make ECHO feel like a home arcade worth bringing into a real adult home - not gamer clutter, not flashy luxury, not a generic gaming gadget.
- Use clear category labeling with the image. The image alone is too easy to misread as a generic game system or screen-based device.

## 2. What changed vs the previous version
- Nostalgia is now the strongest first-touch route and should be treated as the default opening message.
- Across-generations togetherness is the strongest secondary route and should be used more often than premium centerpiece language.
- Plug & play and curated-library value remain strong - but they work best as secondary proof, not the broadest first hook.
- Premium home-bar / centerpiece language should move down the ladder and be used for narrower segments only.
- Explicit category anchors matter more than before because image-only discovery often gets read too generically.

## 3. Message priority ladder
| Priority | Route | Role | Practical use |
|---|---|---|---|
| 1 | Nostalgia | Default opening route | Bring back the classics. Relive arcade nostalgia in a modern home-ready format. |
| 2 | Togetherness | Best secondary route | Connect across generations. Turn hanging out into playing together. |
| 3 | Ease / ownership | Best bridge from emotion to conversion | Plug & play. Easy to start, easy to come back to, easy to own. |
| 4 | Curated library value | Rational support route | All the favorites in one place. No hunting, no setup hassle. |
| 5 | Adult-home fit / quiet premium | Not for Discovery openers | Made for real homes, built to last, looks right in the room. Use only in retargeting, bottom-funnel, or narrower warm audiences. |

**Campaign-confirmed (April 2026):** The Easter Meta Ads campaign validated this ladder. The text closest to Priorities 1-2 with the strongest structure produced the best CPC ($0.18). A warranty/proof-first opener (Priority 5 territory) produced the worst CPC ($0.22) and low CTR. Stacking multiple priorities in one text produced high volume but at the highest CPM ($6.49).

### Proven Discovery text structure
The best-performing Discovery text followed this template:
1. **Line 1-2:** One sharp emotional hook. One idea only (nostalgia or togetherness). Concrete enough to stop the scroll.
2. **Line 3:** Category + proof. Name what ECHO is (home arcade) and include one proof point (149 games, gameplay styles, etc).
3. **Line 4:** Ease bridge. Plug and play, no setup, one machine.
4. **Line 5:** Short CTA.

Rules for this template:
- Do not stack multiple emotional routes in one text.
- Do not open with warranty, support, or durability. These belong in retargeting.
- Do not open with bar-culture framing in broad campaigns.

## 8. Headline rules
- Anchor + Nostalgia -> Home arcade, classic vibes.
- Nostalgia + Home -> Bring the arcade home.
- Anchor + Togetherness -> Retro arcade for game night.
- Ease + Anchor -> Plug & play home arcade.
- Library + Nostalgia -> The classics, ready again.
- Home place + Anchor -> Game room, meet your new favorite.
- Adult-home fit + Nostalgia -> Grown-up nostalgia at home.

Practical rules:
- 4-8 words is ideal.
- Rough target: 25-45 characters when possible.
- No semicolons.
- If the line needs two commas, it is probably too long for Discovery.
- Do not stack nostalgia + ease + premium + library in one line.
- Choose one driver per headline.

## 11. Creative coordination notes for designers
- If the image is static, the headline must clarify the category - home arcade, bartop arcade, touchscreen arcade for home, or all-in-one arcade.
- Do not pair a static hero image with a vague line like 'big fun' or 'premium centerpiece' and expect the category to land.
- If using video, show the machine working in the first seconds - taps, game motion, player reaction, room context.
- Prioritize room-realism over showroom gloss - den, basement lounge, home bar, game room, cabin, lake-house feel.
- Show close-up proof of screen, controls, finish, and materials in support frames.

## 12. Fast QA checklist for ad managers
- One idea per line.
- Lead with nostalgia, togetherness, or ease - not specs.
- Use explicit category labeling if the visual alone is ambiguous.
- Keep premium quiet, adult, and grounded.
- No gamer jargon, no console language, no flashy luxury.
- Do not open with warranty, support, or durability language in Discovery.
- Do not stack multiple message routes in one text.
- Ask: does this feel like a home product worth owning for years?
- Use only short dash '-' in final text.

## 13. A/B testing methodology reference
When testing headlines, game screens, image bases, or formats, use the sequential isolation method established in April 2026:

- Test one variable at a time. Hold all others constant.
- Run each ad in its own ad set with equal daily budgets to prevent algorithmic bias.
- Statistical threshold: 0.5pp minimum detectable effect, 95% confidence, 80% power = ~16,770 impressions per ad (at planning CPM of $5.50).
- Primary decision metric: CPC. Secondary: CTR.
- Kill rule: if one ad has 2x the CPC of another after both reach threshold impressions, kill it and reallocate.
- Recommended round sequence: creative concept - message refinement - game screen - format (static vs video).
- Adjust impression requirements for platforms with different CPMs, but keep the same statistical framework.
