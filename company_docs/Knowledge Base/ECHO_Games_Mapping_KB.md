# ECHO Games Mapping - Genre and Reference Title Index

Working reference for the ECHO game library.
Maps each of the 149 ECHO titles to a common game genre and, where appropriate, a well-known reference title that describes the style of play.

## How to use this document
Use this file when you need to:
- describe the ECHO library by gameplay style rather than by title name
- identify which popular game experiences ECHO covers
- group ECHO titles into recognizable genre clusters for messaging, briefs, or content
- check which reference titles are relevant when selling by "feels like" language

**Critical rule:** All 149 games are JVL's own proprietary titles. No licensed third-party game names exist in the library. Reference titles listed here are for internal mapping only - they describe the style of play the TA would recognize. Do not use trademarked game names (Pac-Man, Donkey Kong, Galaga, etc.) in any ECHO marketing, even in descriptive form. Card game names (poker, blackjack, solitaire) and generic game-type words (mahjong, sudoku, pinball, darts, checkers, backgammon, bingo, dominoes) are generic and safe to use.

## Structure
- **ECHO Title** - the actual game name inside the machine
- **ECHO Category** - the category label used on the machine itself (Action, Cards, Puzzle, Strategy, Quiz)
- **Game Genre** - the common gameplay genre this title maps to
- **Reference Titles** - well-known games or game types the TA would recognize as similar (internal use only)

## What this mapping means for messaging

The library's strongest sellable clusters are:
- **Cards** - poker, blackjack, solitaire. The TA's strongest card-game vocabulary matches perfectly. Safe to name directly.
- **Puzzle** - the deepest genre. Covers many familiar casual-play styles. Describe as puzzles, brain teasers, match games, mahjong, sudoku, and similar.
- **Strategy / board-game classics** - backgammon, checkers, dominoes, bingo, battleships. Highly familiar to the TA's social-play habits. Safe to name directly.
- **Sports / bar games** - billiards, darts, bowling, shuffleboard, air hockey. Maps directly to the home-bar and game-room context. Safe to name directly.
- **Quiz / word games** - trivia and word puzzles. Familiar to the TA's board-game and party-game habits. Describe as trivia and word games.
- **Shoot 'em up / score chasers** - the closest match to classic arcade-memory titles. Describe by gameplay feel (score chasers, classic shooters), not by trademarked names.

### Working rule for library messaging
Sell by gameplay style and familiar game-type words. The strongest safe language includes: poker, blackjack, solitaire, trivia, word games, puzzles, mahjong, sudoku, darts, billiards, bowling, checkers, backgammon, pinball, brain teasers, score chasers, racing, and bar-sports classics.
