## 1. How to use this pack

Use these templates when you need to turn the current research base into execution.
Do not rewrite the strategy from scratch each time.

Best workflow:
1. Pick the business goal.
2. Pick one primary message route only.
3. Pick one main audience mode only.
4. Pick the proof stack that best fits the product page / ad / asset.
5. Fill the relevant template below.
6. Run the QA checklist before review.

Non-negotiable working rules:
- Lead with nostalgia or togetherness before heavy proof, unless the task is bottom-funnel.
- Use clear category language - home arcade, tabletop arcade, countertop arcade machine, all-in-one arcade machine.
- Avoid gamer-jargon, luxury-for-status tone, and novelty-gadget framing.
- The product must feel adult, grounded, and worth owning.
- Emotion opens interest. Proof closes the sale.

## 2. Fast message hierarchy

Use this priority order unless a brief clearly calls for a narrower route.

### Priority 1 - Nostalgia
Use when you need the strongest first hook.

Core thought:
- Bring back the classics in a modern home-friendly way.

Signals:
- classics
- arcade memories
- familiar games
- relive what you loved

### Priority 2 - Togetherness
Use when the goal is broad family/social appeal.

Core thought:
- Easy shared fun across generations - kids, adults, friends, guests.

Signals:
- reconnect
- share
- bring people together
- something everyone can try

### Priority 3 - Plug and play ease
Use when buyers may fear complexity, setup burden, or hassle.

Core thought:
- Easy to set up, easy to understand, easy to keep using.

Signals:
- plug and play
- ready in minutes
- all-in-one
- no complicated setup story

### Priority 4 - Curated game library value
Use when you need to justify what is inside the product.

Core thought:
- A built-in collection of classic-style games in one machine.

Signals:
- all the favorites in one place
- classic game library
- one machine, many games

Note: All 149 games are JVL's own proprietary titles. No licensed third-party game names exist. Sell by gameplay style (maze chase, card games, puzzlers, score chasers), not by implying recognizable licensed brands. Card game names (poker, blackjack, solitaire) are generic and safe to use. Do not reference trademarked game titles (Pac-Man, Donkey Kong, Galaga, etc.) anywhere, even in descriptive form.

### Priority 5 - Quiet premium / adult-home fit
Use later in the sequence, or for narrower higher-intent audiences.

Core thought:
- Well-made, tasteful, durable, good in the home.

Signals:
- adult-home fit
- built to last
- grounded quality
- well-made, not flashy

Do not make this the broad first-touch route unless the audience is already warm.

- Does the tone feel adult, warm, and grounded?
- Does the ad avoid gamer-jargon and novelty-tech energy?

## 15. A/B testing methodology (established April 2026)

Use this framework for any paid ad test across channels.

### Principles
- Test one variable at a time. Hold all others constant.
- Run each ad variant in its own ad set with equal daily budgets. Do not rely on platform auto-allocation (e.g., Meta Advantage+), which will starve slower starters before they reach statistical significance.
- Use the sequential isolation method: test the biggest strategic question first, then refine the winner.

### Statistical thresholds
- Minimum detectable effect: 0.5 percentage points (CTR difference)
- Confidence level: 95%
- Statistical power: 80%
- Required impressions per ad: ~16,770 (at planning CPM of $5.50, derived from Easter 2026 campaign data)
- Adjust impression requirements when CPMs differ by platform or market.

### Decision rules
- Primary metric: CPC (cost per click)
- Secondary metric: CTR
- Minimum threshold before judging: each ad must reach its required impression count
- Kill rule: if one ad has 2x the CPC of another after both reach threshold, kill the underperformer and reallocate budget
- Early kill: if the gap is 2x CPC after 70%+ of threshold impressions, it is unlikely to recover

### Recommended round sequence
1. Creative concept test (image base + natural message pairing) - 3 ads
2. Message refinement (alternate emotional angle within winning concept) - 2 ads
3. Game screen test (e.g., poker vs score chaser) - 2 ads
4. Format test (static vs video) - 2 ads

Each round runs until all ads reach their impression threshold plus a platform learning-phase buffer (2 days for 3-ad rounds, 1 day for 2-ad rounds on Meta).

### Constants
Unless a test specifically isolates these, hold constant across all variants:
- Secondary line: "Home arcade - 149 games"
- CTA: "Bring It Home"
